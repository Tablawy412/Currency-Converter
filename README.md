# Currency Converter (USD to EGP)

A sleek, modern, and fully responsive web application that allows users to convert currencies between US Dollars (USD) and Egyptian Pounds (EGP) in real-time.

---

## Features

* **Real-Time Calculation:** Converts values instantly as you type without needing a page refresh or a dedicated convert button.
* **Fully Responsive Design:** Optimized for all screen sizes, ensuring a seamless user experience from desktop monitors to mobile phones.
* **Modern UI/UX:** Built with clean input fields, custom focus states, smooth transitions, and elegant box shadows.
* **Input Validation:** Prevents negative numbers and non-numeric characters to ensure accurate calculations.

---

## Tech Stack

* **HTML5:** Semantic structure for optimal accessibility and document hierarchy.
* **CSS3:** Custom styles leveraging modern Flexbox layout, Box Model principles, and responsive design techniques.
* **JavaScript (ES6):** Live DOM manipulation and dynamic input event handling.
